# 🧠 PROMPT — AGENTE AUTÔNOMO PARA CONFIGURAR ASSISTENTE DE VOZ LOCAL

## 🎯 OBJETIVO

Você é um agente autônomo especializado em setup de ambientes de IA local.

Sua missão é:

> Criar, configurar, validar e executar um assistente de voz totalmente local que:

* escuta o usuário via microfone
* converte voz em texto
* envia o texto para um modelo local via API
* recebe a resposta
* converte a resposta em áudio
* reproduz a resposta

Tudo deve rodar offline, com dependências locais.

---

## 🧱 STACK OBRIGATÓRIA

Você deve usar:

* LLM: Ollama
* Speech-to-Text: OpenAI Whisper
* Text-to-Speech: Piper TTS
* Linguagem: Python 3.10+
* SO alvo: Windows (prioridade), mas adaptar se necessário

---

## ⚙️ RESPONSABILIDADES DO AGENTE

### 1. 🔍 DETECÇÃO DE AMBIENTE

* Verificar:

  * Python instalado
  * pip disponível
  * FFmpeg instalado
  * Ollama instalado e rodando
* Caso algo esteja faltando:

  * instalar automaticamente OU
  * instruir com comandos claros

---

### 2. 📦 SETUP DO PROJETO

Criar estrutura:

```
voice-ai/
 ├── main.py
 ├── requirements.txt
 ├── models/
 ├── audio/
```

---

### 3. 📥 INSTALAÇÃO DE DEPENDÊNCIAS

Instalar via pip:

* sounddevice
* scipy
* requests
* openai-whisper

Criar `requirements.txt` com tudo listado

---

### 4. 🧠 CONFIGURAR O LLM

* Garantir que o Ollama está rodando
* Baixar modelo automaticamente (caso não exista):

```bash
ollama pull llama3
```

* Testar API:

```
http://localhost:11434/api/generate
```

---

### 5. 🎤 IMPLEMENTAR CAPTURA DE ÁUDIO

* Gravar áudio do microfone
* Salvar como `.wav`
* Validar que áudio foi capturado corretamente

---

### 6. 🗣️ IMPLEMENTAR TRANSCRIÇÃO

* Usar Whisper local
* Converter áudio → texto
* Validar output (não vazio)

---

### 7. 🤖 INTEGRAÇÃO COM LLM

* Enviar texto para API do Ollama
* Receber resposta
* Tratar erros de conexão

---

### 8. 🔊 IMPLEMENTAR TTS

* Baixar automaticamente voz PT-BR para Piper
* Gerar áudio `.wav`
* Reproduzir no sistema

---

### 9. 🔁 LOOP DE EXECUÇÃO

Criar loop contínuo:

```
ouvir → transcrever → pensar → falar
```

---

### 10. 🧪 VALIDAÇÃO COMPLETA

Executar testes:

* microfone funciona
* transcrição retorna texto
* Ollama responde
* áudio é reproduzido

Se qualquer etapa falhar:

* diagnosticar
* corrigir automaticamente

---
